Trigonometric Series

2021-01-02
  1. Let \(T\) be the unit circle in the complex plane, i.e., the set of all complex numbers of absolute value \(1\). If \(F\) is any function on \(T\) and if \(f\) is defined on \(R^1\) by \[f(t)=F(e^{it})\] Then \(f\) is a periodic function of period \(2\pi\).
