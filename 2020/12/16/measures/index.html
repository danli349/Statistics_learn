Measures

2020-12-16

  1. The empty set is \(\varnothing\). A collection \(\tau\) of subsets of a set \(X\) is said to be a topology in \(X\) if \(\tau\) has the following properties: (i) \(\varnothing\in\tau, X\in\tau\). (ii) If \(V_i\in\tau, i=1,2,\cdots,n\), then \(V_1\cap V_2\cap \cdots\cap V_n\in\tau\). (iii) If \(\{V_a\}\) is an arbitrary collection of members of \(\tau\), then \(\bigcup_aV_a\in\tau\). If \(\tau\) is a topology in \(X\), then \(X\) is called a topological space, and the members of \(\tau\) are called the open sets in \(X\). If \(X\) and \(Y\) are topological spaces and if \(f\) is a mapping of \(X\) into \(Y\), then \(f\) is said to be continuous provided that \(f^{-1}(V)\) is an open set in \(X\) for every open set \(V\) in \(Y\).

  2. A collection \(\mathfrak M\) of subsets of a set \(X\) is said to be a \(\sigma\)-algebra in \(X\) if \(\mathfrak M\) has the following properties: (i) \(X\in\mathfrak M\) (ii) If \(A\in \mathfrak M\), then \(A^c\in\mathfrak M\), where \(A^c\) is the complement of \(A\) relative to \(X\). (iii) If \(A=\bigcup_{n=1}^{\infty}A_n\) and if \(A_n\in\mathfrak M, n=1,2,\cdots,\) then \(A\in\mathfrak M\). If \(\mathfrak M\) is a \(\sigma\)-algebra in \(X\), then \(X\) is called a measurable space and the members of \(\mathfrak M\) are called the measurable sets in \(X\). If \(X\) is a measurable space, \(Y\) is a topological space and \(f\) is a mapping of \(X\) into \(Y\), then \(f\) is said to be \(measurable\) provided that \(f^{-1}(V)\) is a measurable set in \(X\) for every open set \(V\) in \(Y\).
