How to get the Cumulative distribution function of Normal distribution

2020-08-16

The Cumulative distribution function of Normal distribution is \[\Phi(x)=\frac{1}{\sqrt{2\pi}\sigma}\int_{-\infty}^{x} e^{-\frac{1}{2}(\frac{t-\mu}{\sigma})^2}dt\] We can move the distribution to \(\mu=0\), which will not change the Cumulative distribution function.

\[\Phi(x)=\frac{1}{\sqrt{2\pi}\sigma}\int_{-\infty}^{x} e^{-\frac{1}{2}(\frac{t}{\sigma})^2}dt\]

The \(\Phi^2(x)\) is easier to calculate using the Polar coordinate system, *when \(x<0\) \[\begin{align} \Phi^2(x)&=\Biggl(\frac{1}{\sqrt{2\pi}\sigma}\int_{-\infty}^{x} e^{-\frac{1}{2}(\frac{t}{\sigma})^2}dt\Biggl)^2\\ &=\frac{1}{2\pi\sigma^2}\int_{-\infty}^{x} e^{-\frac{1}{2}(\frac{t}{\sigma})^2}dt\int_{-\infty}^{x} e^{-\frac{1}{2}(\frac{s}{\sigma})^2}ds\\ &=\frac{1}{2\pi\sigma^2}\int_{-\infty}^{x}e^{-\frac{t^2+s^2}{2\sigma^2}}dtds\\ &=\frac{1}{2\pi\sigma^2}\int_{x}^{+\infty}e^{-\frac{t^2+s^2}{2\sigma^2}}dtds\\ &=\frac{1}{2\pi\sigma^2}\int_{0}^{\frac{\pi}{2}}d\theta\int_{x}^{+\infty}e^{-\frac{r^2}{2\sigma^2}}rdr\\ &=\frac{1}{4\sigma^2}\int_{x}^{+\infty}e^{-\frac{r^2}{2\sigma^2}}rdr\\ &=-\frac{1}{4}\int_{x}^{+\infty}e^{-\frac{r^2}{2\sigma^2}}d(-\frac{r^2}{2\sigma^2})\\ &=-\frac{1}{4}(0-e^{-\frac{x^2}{2\sigma^2}})\\ &=\frac{1}{4}e^{-\frac{x^2}{2\sigma^2}} \end{align}\]

Then \(\Phi(x)=\sqrt{\frac{1}{4}e^{-\frac{x^2}{2\sigma^2}}}=\frac{1}{2}e^{-\frac{x^2}{4\sigma^2}}\)

*when \(x\ge0\), \(\Phi(x)=1-\Phi(-x)=1-\frac{1}{2}e^{-\frac{x^2}{4\sigma^2}}\)
